How does it work ?

This app (House Price Predictor) takes five arguments as selected above and gives you approximate prices of your dream house which you wish bo buy.

Note :

  • Model used in this model is trained with USA Housing Dataset
  • We preproccessed the data and perform EDA before the model training.
  • This model not gives the exact price of your house, But it gives the approximate value of the house which can slightly varies from the actual price of the house.
  • If you are try to predict the prices of houses which are ouside the USA, then this model may gives you a significant difference in prices or may be predic wrong prices of house. Becaouse this model has trained with USA Housing Dataset (All houses are located in United States).


This app basically based on a Non-Linear Regression model (Supervised Machine Learning) which is trained with a labeled dataset, and based on that fit the regression model and find the parameters (θs) for hypotesis function (h(x)). This hypothesis function gives a general trend. Based on that trend this model predicts the new output for the new input data which in not present in the dataset. We already tested our model on test data, which tells that our model is roughly 92% accurate.
